Client Notes:
---------------------------------------------------------------------------
To connect to a server go into file->connect.  Then to do anything with the 
spreadsheets go to file->new, open, rename, etc.
---------------------------------------------------------------------------
Server Notes:
---------------------------------------------------------------------------

Use the 'make' command in the terminal to build the server and then use the 
'make run' command to run it.
